# mysqldumpdbs

Dump all MySQL databases in separate files

## Usage

#### Option 1

Run the script from remote host (Github)

```bash
curl -sL http://git.io/vvyfd | bash -
```

#### Option 2

Clone the repository and run locally

```bash
git clone https://github.com/IvanDokov/mysqldumpdbs.git
cd mysqldumpdbs
./mysqldumpdbs.bash
```

## Ignored databases

* mysql
* performance_schema
* information_schema
